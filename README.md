# Thesis — Demographic Dividend Reversed: Investing Around a Shrinking Workforce
**2026 Summer Internship Program**

**Checkpoint 1 Submission**

## Project Overview

Birth rates are collapsing across developed and middle-income nations. This
project investigates the investment implications of that shift — automation,
immigration policy, pension system stress — and tests which industries
structurally shrink versus benefit from a tightening labor market over the
coming decades.

**Stance tested this checkpoint:** Demographic contraction is creating real,
measurable labor scarcity in aging economies (US, Japan, South Korea). The
investment response — AI-driven automation substituting for labor — is still
in its early innings. The market has not yet fully priced this in, because
the current wave (generative AI / agentic automation, 2023 onward) is
mechanically different from the prior robotics-era automation cycle. **The
opportunity is forward-looking, not backward-confirming.**

This checkpoint builds the foundational pipeline — data ingestion,
hypothesis testing, visualization, and a documented verdict — that later
checkpoints will extend with immigration-policy variables, pension/fiscal
stress indicators, and a broader country set.

## Methodology

1. **Demographic mechanism check** — pull annual age-dependency ratios
   (2005–2024) for the US, Japan, and South Korea from the World Bank API.
   Run a linear regression on each country's trend to confirm the
   working-age population is structurally shrinking relative to dependents.

2. **Investment response check** — pull daily prices for an
   automation/robotics ETF basket (ROBO, BOTZ) and a labor-intensive/
   domestic-consumption basket (XRT retail, ITB homebuilders) via yfinance.
   Build equal-weighted, indexed-to-100 baskets for each.

3. **Timing test** — split basket performance at the AI-wave inflection
   point (January 2023, following ChatGPT's late-2022 public release).
   Compare the automation-vs-labor return spread pre- vs. post-split.

4. **Significance test** — Welch's t-test on annual spread values, pre- vs.
   post-AI-wave.

**Verdict logic:** the stance is marked `supported_early_innings = True`
only if the post-2023 spread is (a) positive and (b) larger than the
pre-2023 spread — i.e., automation is not just outperforming, but
outperforming *more* since the AI wave began. This directly operationalizes
the "early innings, not yet priced in" framing rather than a simple
"did automation beat labor-intensive stocks" binary.

## Limitations

- Correlation between demographic trend and basket performance does not
  establish causation; confounders include Fed policy, the broader AI hype
  cycle, USD strength, and sector-specific catalysts unrelated to
  demographics.
- The post-AI-wave window (2023–2024) is short, limiting statistical power
  — expected to strengthen as more data accumulates.
- ETF baskets are imperfect proxies for "automation" and "labor-intensive"
  exposure.
- A single dependency-ratio threshold (55) is used to classify regimes;
  sensitivity testing is planned for the next tier.
- Only three countries are analyzed; broader middle-income coverage is
  planned for a later checkpoint.
